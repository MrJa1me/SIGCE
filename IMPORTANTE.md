# SIGCE — Demo en línea

Este proyecto **SIGCE** (Sistema Integrado de Gestión de Check-in en Aduanas) se encuentra desplegado y en ejecución en el **servidor privado de Nicola Guajardo**.

## Acceso

Puedes acceder a la aplicación por internet en:

**https://sigce.mrjaime-glb.uk**

## Disponibilidad

La demo estará **disponible hasta el 17 de julio de 2026**. Después de esa fecha, el servicio en este servidor puede dejar de estar activo.

## Sobre el código en GitHub

La versión publicada en GitHub corresponde al proyecto original del equipo, pero **fue adaptada y modificada** para poder ejecutarse en el servidor privado mencionado (configuración Docker, despliegue, variables de entorno, proxy, etc.).

Si necesitas desplegarla en otro entorno, revisa también `DEPLOY.md` y los archivos en la carpeta `deploy/`.
